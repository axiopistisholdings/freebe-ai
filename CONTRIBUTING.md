# Contributing

Thank you for your interest in Freebe AI.

This repository does **not** accept code contributions.

## Why?

Freebe AI is a proprietary system, and production code and infrastructure are
maintained privately.

## Partnerships

We welcome **API-based partnerships** with independent AI developers and teams.

If you are interested in collaboration or integration, please contact:
fusion@freebeai.com

Pull requests, issues requesting features, or implementation changes
will not be accepted.
